# Response Format Guide

When answering code lookup questions, use this format:

## Error Code Response Format

```
Code: [CODE]
Category: [CATEGORY]
Description: [DESCRIPTION]
```

## Status Code Response Format

```
Status: [CODE]
Meaning: [MEANING]
```

## Example

For error code E002:
```
Code: E002
Category: Permission
Description: Permission denied - insufficient access rights
```
